About SmartContactManager

Your ultimate solution for managing contacts efficiently, securely, and effortlessly.

Project Overview

SmartContactManager is a full-stack web application designed to help users organize, store, and manage their contact information securely. Built with modern technologies, it provides a seamless experience for both individual users and teams.

Key Features

Contact Management

Add, edit, delete, and search contacts easily.

Secure Authentication

Login/signup with Spring Security encryption.

Database Storage

MySQL database for persistent data storage.

Photo Upload

Upload profile pictures for each contact.

Search & Filter

Find contacts quickly with advanced search.

User Friendly

Clean interface designed for easy navigation.

Technology Stack

Java Spring Boot Spring Security JPA / Hibernate MySQL HTML / CSS / Thymeleaf JavaScript

Built With Passion

This project demonstrates best practices in Spring Framework development, including secure authentication, database management, and responsive UI design.
